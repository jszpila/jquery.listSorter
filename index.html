jQuery & HTML5 Custom List Sorter

What It Does

With this plugin, you'll be able to make lists sortable according to the list item's custom data attributes.

Usage

On DOM ready, use any of the following methods:

  • $('.myContainerClass').listSorter(); // for all instances for a class
  • $('#myContainerElement').listSorter(); // for a single element by ID
  • $('#myContainerElement1, #myContainerElement2').listSorter(); // for container elements with different IDs

Requirements

  • jQuery (duh) - developed with 1.4.2
  • An element with the class .sortFilters that contain links used to control filtering:
    • Links that should trigger filtering must have the class .sortFilter
    • Filter control links must contain two data fields: data-SortByField and data-SortDirection
      • data-SortByField must contain the name of the data field attribute of the list items in the list you wish to sort.
      • data-SortDirection must contain either "asc" or "desc" as it's value.
  • A list with the class .sortList
  • Items in the of the sortable list must contain a data field attribute that matches those specified in the filters.

Notes:

  • Alphabetical sort is accomplished by designating an empty string in the data-SortByField.
  • Contents of sort fields should be of same type.

Generated Classes

  • .activeSortFilter - is applied to the link that applied the current sort.
  • .sort_[FIELD NAME]_[SORT DIRECTION] - is applied to the list and changes on each sort, where [FIELD NAME] is the field sorted by, and [SORT DIRECTION] is the sort direction.
  • .sortRank_[FIELD NAME]_[FIELD VALUE] - is applied to list items.

Notes:

  • When sorting alphabetically, the value of [FIELD NAME] used in the generated classes will be "alpha".
  • When sorting alphabetically, the value of [FIELD VALUE] used in the generated classes with be the uppercase value of the first letter.

Example #1

Single list.

Sort: alphabetical (ascending), alphabetical (descending), data-Rank (ascending), data-Rank (descending)
  • A, rank 1
  • Z, rank 2
  • C, rank 3
  • J, rank 6
  • N, rank 7
  • F, rank 9
